# Backend

This folder is reserved for the API service.

Suggested layout:
- src/config: environment and app configuration
- src/routes: HTTP routes
- src/controllers: request handlers
- src/services: business logic
- src/middleware: auth/logging/error handling
- src/db: database access
- tests: backend tests

Add your chosen backend stack here when ready (Express, NestJS, Fastify, etc.).
